# Unloop

Break the scroll. Reclaim your time.

Unloop is a behavioral transformation platform designed to help users reduce screen addiction, track focus, and reclaim attention.

## Features

- Screen time tracking
- Focus sessions
- Recovery system
- Streak tracking
- Analytics dashboard
- Widgets
- Privacy-first design

## Tech Stack

- TanStack Start
- React
- TypeScript
- Vite
- Capacitor
- Android
- Vercel
